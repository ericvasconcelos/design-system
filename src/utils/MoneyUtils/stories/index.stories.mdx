# MoneyUtils

Conjunto de métodos para manipulação de dinheiro.

## format

O método `format()` formata um número para uma string.

### Sintaxe

```MoneyUtils.format(entry)```

#### Parâmetros

- `entry`: número a ser formatado.

#### Valor retornado

String formatada.
