# QueryUtils

Conjunto de métodos para manipulação de query string.

## toString

O método `toString()` transforma um objeto em uma query string.

### Sintaxe

```QueryUtils.toString(object)```

#### Parâmetros

- `object`: objeto com os valores da query string.

#### Valor retornado

Query string.

## toObject

O método `toObject()` transforma uma query string em um objeto.

### Sintaxe

```QueryUtils.toObject(string)```

#### Parâmetros

- `string`: query string.

#### Valor retornado

Objeto com chave e valor dos elementos da query string.
