# ObjectUtils

Conjunto de métodos para manipulação de objetos

## clean

O método `clean()` limpa chaves que possuem valores vazios, null ou undefined.

### Sintaxe

```ObjectUtils.clean(object)```

#### Parâmetros

- `object`: objeto para ser limpado.

#### Valor retornado

Novo objeto com as chaves removidas.
