# NumberUtils

Conjunto de métodos para manipulação de números.

## sanitize

O método `sanitize()` remove caracteres que não são dígitos.

### Sintaxe

```NumberUtils.sanitize(number)```

#### Parâmetros

- `number`: string.

#### Valor retornado

String com apenas dígitos.
